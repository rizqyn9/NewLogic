To do a headless build from the command line, start the Unity Editor with the following parameters:
-quit -batchmode
-projectPath "path to project folder"
-headlessPath "path to destination folder"
-executeMethod HeadlessBuilder.BatchBuild

If you have multiple profiles and would like to build a profile that is not the default, add:
-headlessProfile "name of profile"

Note:
Optionally, you may use the -buildTarget command line parameter. Refer to the Unity Documentation for the possible values.
This will of course override the build target chosen in the Headless Builder settings.

Note:
Any warnings or errors will be printed to the Unity Editor log.
